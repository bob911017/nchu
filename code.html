    R語言

    如何安裝packages
    1. 連上網路後，輸入install.packages("套件名稱")即可安裝某套件
    2. 在安裝某套件後，輸入library(套件名稱)可引入該套件，之後才可使用它的功能
    3. 連上網路後，輸入update.packages()可更新已安裝的所有套件
    基本功能
    • 叫回過去打過的程式碼
      history()

    • 查詢已擁有的套件
      search()

    • 查詢已擁有的套件位置
      searchpaths()

    • 顯示套件的範例
      example(gtrends)

    help相關指令
    • 知道指令名稱
      ?plot

    • 不知道指令正確名稱

    • 函數名稱中有plot
      apropos("plot")

    • 尋找plot在哪個package
      find("plot")
      getAnywhere("plot")

    • 查詢plot參數
      args("plot")

    資料輸出與輸入路徑
    • 檢視資料夾底下有什麼檔案
      dir()

    • 設定工作資料夾路徑:
      setwd("D:\\")

    讀檔與寫檔
    • 以逗號分隔讀入資料
      read.csv("檔名")

    • 以Tab分隔資料
      read.table("檔名")

    • 設定分隔符號
      read.table("檔名",sep=",")
      read.table("檔名",sep="\t")

    • 將標題讀入，加上row.names可把文字資料左移，方便只處理數字資料
      read.table("檔名",header=T,row.names=1)

    • 寫檔:
      write.csv("檔名")
      write.table("檔名")

    • 去掉Tab產生的雙引號
      write.table("檔名",quote=F)

    • 去掉橫軸及縱軸標題
      write.table("檔名"row.names=F,col.names=F)

    • 利用getwd()可確認目前工作位置，設定好工作位置後，寫檔和讀檔就不用寫了
      getwd()

    計時(time)
    from time import clock,time
    
tstart=clock()
    
要執行的程式碼
    
tend=clock()
    
print tstart,tend,tend-tstart

    這樣的打法跟只用import time的不同在於，我特地呼叫出time函數庫裡的time和clock，因此tstart=clock()那裏就不必特別打成tstart=time.clock()
