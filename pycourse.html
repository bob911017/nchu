  • 基礎概念

    • 概念一

      • 程式在運行的時候，進行的是數學上的運算，因此想要利用變數來協助編寫程式碼的時候，需要指定一個數值給變數。指定的方式就是利用「=」這個符號。範例如下

      a=2

      範例代表的意思即「把2這個值指定給a這個變數」。另一方面，可以利用另一個變數來與這個變數a做連結，例如b=a。此時b和a會被程式視作一體的，當b的數值更動時就會影響a，反之亦然。

    • 概念二

      • 利用概念一，可以衍伸出類似加減乘除的計算方式。範例如下

      a=a+3

      意思是「a+3得到的值指定給a這個變數」。也就是說，假如原本a的值為2，那麼加上3之後就會得到5。程式把5這個值指定給a這個變數。

  • 資料的類型(Type)

    • 儲存資料有很多種方式，包含數字、數列、字串等等。不同類型的資料通常不能直接做疊加或運算，需要將待處理的資料轉換成相同的類型，才能一起運算。資料之間的類型轉換方式如int(1.1)，就能把原本是浮點數的1.1轉換成整數形式的1。
    以下我整理一下比較常用的幾種type。

    1. 數字

      2. 數字分為int(integer整數)、float(floating-point number浮點數)和complex(complex number複數)。這邊因為複數沒辦法對應相應的實數，所以先主要討論整數和浮點數的部分。
      我以10為例，整數形式int(10)為10，而浮點數形式float(10)為10.0，接著以10.6為例，整數形式int(10.6)為10而浮點數float(10.6)為10.6。初學者容易犯的錯就是把兩個整數做運算後，得出的結果並非自己所想的。範例如下:

      程式碼
      print 3/6

      顯示為
      0

      一般而言雖然我們知道3/6等於0.5，但在這邊程式認知你的運算都是整數的類型，因此他把答案0.5取整數，就是程式所顯示的0了。解決的方法很簡單，只要把運算改成3.0/6或3/6.0或是3.0/6.0就能顯示0.5了。

    2. 數列與矩陣

      3. 可以處理數列的形式分為list和array這兩種，之前用np.arange叫出來的就是array的形式(雖然Type裡寫float64或int64，不過這之後再說)。這兩者都是儲存多個資料的方式，建立list的方法只要打上a=[0,1,2]或是a=range(3)就是一個list了!另一方面，array的話，幾乎用numpy建立出來的數列或矩陣都是這個形式的資料類型。

    3. 字串string

      4. 字串就是用來儲存文字資料的一種資料形式，使用方法只要在要儲存的文字兩側放上「'」即可，如「'文字'」

      範例: print 'hello world!'

  • 判斷條件(True or False)

    • 判斷條件是通過判斷一個或多個語句的結果為True或False，來決定執行的程式碼。判斷式中大概包含等於(==)、不等於(!=)、大於(>)、小於(<)和大於等於(>=)等種類。
    當符合判斷式條件的時候，Python會給出True的回應，反之則為False。大家可以用以下程式碼進行嘗試

    程式碼
    print 1==1
    print 1!=1

    輸出為
    True
    False

    另外如果有多個條件的話，則可以使用聯集(or)或是交集(and)的概念，範例如下

    print 1==1 or 1!=1

    結果為
    True

函式(function)

  • 建立函式(def)

    • 建立函式算是滿好用的一個指令，如果在程式碼中有一很常用到的計算式的話，就能利用def讓整個程式碼看起來更簡潔。以下為範例

    import matplotlib.pyplot as plt
    import numpy as np
    #建立函式
    def velocity(x1,x2):
        y=x1*x2
        return y

    t=np.arange(0.0,10.0,0.2)
    v=velocity(9.8,t)
    plt.plot(t,v,'-o')

    輸出的結果會是一張v=at的圖。函式較熟悉的形式莫過於f(x)=y，意思就是給定(input)一個值x，經過f()這個方程式轉換後，形成一個y的值。
    上面範例中v=velocity(9.8,t)進行的動作包含
    1. 將9.8和t分別代入velocity(x1,x2)中的x1和x2，
    2. 經過方程式中的y=x1*x2計算後得出一個y值
    3. 把計算後得出的y值回傳給v=velocity(9.8,t)這條方程式中的v


  • 函式庫的引入與使用

    • 在python的世界裡，已經有許多人編寫好的函式可供引入並使用，如SciPy.org這裡面提供的，其中包含各種數學運算或繪圖等不同的功能。以下就來說明如何使用

    1. 載入函式庫並取代稱

      2. 使用的方式為「import 函式庫名稱 as 代稱」這樣的形式

      範例:
       import numpy as np

      打上如上列式子的程式碼後，就完成引入函式庫的動作了。此時當我們要引用函式庫numpy中的函式pi的功能時，只要打np.pi。當然覺得取代稱很麻煩也可以直接打import numpy就好，而引入函式的形式就變為numpy.pi。

    2. 第二個引入函式庫的形式


      3. 這個引入函式庫的方式比較偷懶。表示的方式是

      「from 函式庫 import *」

      範例:from numpy import *

      這個指令會自訂幫你搜尋你打的函式在哪個函式庫中，也就是本來要打np.pi的情況，你只要打pi就可以直接使用。這雖然對編寫者很方便，但會讓電腦多花一些時間尋找你的函式在哪個函式庫，程式執行的速度相比直接告訴他位置會來的慢。

數列

  • 基本操作

    • 建立數列的方式有很多種，包含如[0,1,2]、range(3)以及np.arange(3)這幾種方式。
    range只能建立一個以整數形成的等差數列，如range(0,3,1)
    而np.arange使用上較range靈活，可以建立小數形成的數列，如np.arange(0,3,0.5)
    另外，數列中各個元素的位置順序是從0開始計算的，以下為範例:

    程式碼
    a=[1,2,3,4]
    print a[0]
    print a[1]
    print a[-1]

    輸出為
    1
    2
    4

  • List和Array的使用差異

    • 乘法對於list和array來說有不同的意義。對於list而言，乘法是對自己本身的數列重複幾次的指令，如[1,2,3]這個list去*2的話，就會形成[1,2,3,1,2,3]的list。另一方面，array所形成的數列的乘除法則比較相近於一般認知的乘除，也就是將數列出的每個元素都進行乘除法。以下為範例

    程式碼
    import numpy as np
    list=[1,2,3]
    array=np.arange(0.0,1.21,0.3)
    list*=3
    array*=3
    print 'list='+str(list)
    print 'array='+str(array)

    輸出為
    list=[1, 2, 3, 1, 2, 3, 1, 2, 3]
    array=[ 0. 0.9 1.8 2.7 3.6]

  • List使用上的問題

    • 這東西有點不太好說明。基本上算是程式本身設計上的一些特色(?)，但python同時也有提供相應的解決方法，所以倒也不是什麼大問題。以下為範例

    程式碼
    a=[1,2,3]
    b=a
    b[1]=5
    print 'a='+str(a)+'\n'+'b='+str(b)+'\n'+'---'
    a=[1,2,3]
    c=a[:]
    c[1]=6
    print 'a='+str(a)+'\n'+'c='+str(c)

    輸出為
    a=[1, 5, 3]
    b=[1, 5, 3]
    ---
    a=[1, 2, 3]
    c=[1, 6, 3]

    如果在編寫程式碼的時候，直接將b=a的話，python會把b和a連結為相同的東西，因此在第四行，我將b的第1個元素更換為5的時候，a的第一個元素也會被跟著替換掉。
    如果想避免這樣的情形發生的話，在複製a的資料可以使用像c=a[:]這樣的方法來複製，這樣變數a和c就不會進行連動。

陳述(Statement)

陳述可以被想成程式語言內的獨立元素。簡單的陳述大多以一個句子組成，如a=1，而複雜一點的陳述如for、if和while，則由多條句子所組成。在python裡使用複雜陳述時，必須使用縮排而非特殊字元或關鍵字來決定開始與結束。也就是用4個空白鍵(或是按一下Tab鍵)，來決定陳述的內容包含哪些部分。

  • for迴圈

    • for迴圈是拿來執行固定次數的一個迴圈，執行的次數取決於你給的數列的長短。範例如下:

    程式碼
    a=[1,3,5,7,9]
    for i in a:
        print i
        print 'next'

    輸出為
    1
    next
    3
    next
    5
    next
    7
    next
    9
    next

  • 條件語句(if)

    • 條件語句指的是假設一些條件，符合條件(True)就執行code，不符合(False)就不執行。設定的條件可以不只一個，以下做個範例

    程式碼
    a=[1,2,4,5,3,6]
    for i in a:
        if i<2:
            print str(i)+' matches condition1'
        elif i<4:
            print str(i)+' matches condition2'
        else:
            print str(i) +' other'

    輸出為
    1 matches condition1
    2 matches condition2
    4 other
    5 other
    3 matches condition2
    6 other

    這邊特別注意到其實在a數列中1是同時符合i<2以及i<4的條件，但是這邊程式只顯示符合i<2的結果。這是因為我們設定i<4這個條件是用elif，也就是else if，也就是不符合先前條件且符合elif設定的條件的時候就執行code，如果前面條件符合的話就不執行。而如果有要顯示「1 matches condition2」的話，就把elif i<4:改為if i<4:就行了。

  • 條件循環(while)

    • while語句使用上有點像條件語句結合for迴圈。以下為範例

    程式碼
    i=0
    while i<3:
        print i
        i=i+1

    輸出為
    0
    1
    2

    這組code表示的是當條件i<3符合的時候，就執行print 和i=i+1，一直執行到條件不符合i<3的時候就跳出while之外，執行下一步程式碼。

數學運算

  • 矩陣

    • 矩陣運算

      • 矩陣的運算有兩種指令來運算，一種是把直接建立矩陣來運算，範例如下

      import numpy as np
      a=np.matrix([[1,2],[3,4]])
      b=np.matrix([[1,2],[3,4]])
      c=a*b
      print c

      另一種方式是建立一個array後再用dot的指令進行矩陣運算。範例如下

      import numpy as np
      x=np.array([[1,2],[3,4]])
      y=np.array([[1,2],[3,4]])
      z1=x*y
      z2=np.dot(x,y)
      print 'z1='
      print z1
      print 'z2='
      print z2

      由範例的結果z1和z2我們可以觀察到，前者不使用dot的指令進行相乘的話，僅只有對應位置的元素相乘，而非我們所理解的矩陣乘法。當如後者使用dot的指令進行運算的話則為一般的矩陣運算。

    • 其他矩陣相關運算(Linear algebra)

      • 取行列式值
      np.linalg.det(a)

      取反矩陣
      np.linalg.inv(a)

  • 解常微分程式ODE(函式庫: scipy.integrate.odeint)


    • 範例:解  -cv-kx=ma，阻尼力+彈力=合力

    import scipy.integrate as sci
    import numpy as np
    import matplotlib.pyplot as plt
    m=5 #(kg)
    k=2 #(N/m)
    c=1.5 #(Ns/m)
    ic=[0,5] #initial condition=[x0(m),v0(m/s)]

    def fun(inp,t):
        #x0=inp[0],v0=inp[1]
        out=[inp[1],(-c*inp[1]-k*inp[0])/m]  #out=[x',x'']
        return out

    t=np.linspace(0.0,30.0,101) #(second)
    x=sci.odeint(fun,ic,t)  #sci.odeint(function,intitial condition)
    plt.plot(t,x[:,0]) #x-t graph

  • 動畫(Vpython的使用)

    • 要學習Vpython要怎麼用的話，基本上可以參考Vpython、Vpython Help和VPhysics NTU這些網站就好。裡面也沒什麼太難的技巧，就只是在將各種參數輸入模型裡就行。要特別注意的地方是，當你把vpython的視窗關閉的時侯，附圖中的藍色框框會切換成像是開始鈕的綠色三角形。


    如果要再次執行程式碼的話，只要點下那個三角形，切換成橘色三角形就可以了!
    另外，這邊有其他人做的模型可以提供參考。

    範例程式:

    from visual import *
    import random as rd
    floor = box(pos=vector(0,0,0), length=4, height=0.5, width=4, color=color.blue)
        #設定為方形、長寬高、顏色
    h=helix(pos=floor.pos+vector(0,0.25,0),axis=vector(0,1,0))
        #設定為彈簧形狀
    ball = sphere(pos=vector(0,4,0), radius=1, color=color.red)
        #設定為圓形、半徑、顏色
    arrow1=arrow(pos=ball.pos+vector(0,ball.radius,0), axis=vector(0,-2,0),color=color.yellow)
        #設定為箭頭形狀
    arrow1.pos-=arrow1.axis
    ball.velocity = vector(0,-1,0)
    dt = 0.1

    while 1:
        rate (20) #播放速度，1/20s
        ball.pos = ball.pos + ball.velocity*dt
        arrow1.pos=arrow1.pos+ball.velocity*dt
        ball.color=[rd.uniform(0.4,1.0), rd.uniform(0.4,1.0), rd.uniform(0.4,1.0)]
        h.axis=ball.pos
        if ball.y < ball.radius:
            ball.velocity.y = abs(ball.velocity.y)
        else:
            ball.velocity.y = ball.velocity.y - 9.8*dt
