2016/12/02

  • 函式庫的引入與使用

    • 函式庫包含了各種數學的運算或是繪圖等不同的功能，但是也不要因為偷懶就把所有可能用到的函式庫都載入，這樣會造成程式的負擔，執行速度會減慢，善用別已有的函式庫對寫程式是很有幫助的。

    1. 載入函式庫並取代稱

      2. 使用上就是「import 函式庫名稱 as 代稱」這樣的形式

      範例: import numpy as np
      當然覺得取代稱很麻煩也可以直接打import numpy就好。如果我們要引用函式庫numpy中的函式pi的功能的話，只要打np.pi，如果沒有取代稱的話則打numpy.pi，才可以使用。

    2. 第二個引入函式庫的形式


      3. 這個引入函式庫的方式我個人比較不推薦。表示的方式是

      「from 函式庫 import *」

      範例:from numpy import *

      這個指令會自訂幫你搜尋你打的函式在哪個函式庫中，也就是本來要打np.pi的情況，你只要打pi就可以直接使用。這雖然對編寫者很方便，但會讓電腦多花一些時間尋找你的函式在哪個函式庫，相比直接告訴他位置來的慢。

  • 資料的類型(Type)

    • 儲存資料有很多種方式，包含數字、數列、字串等等。不同類型的資料通常不能直接做疊加或運算，需要將需要處理的資料轉換成相同的類型，才能一起處理。資料之間的類型轉換方式如int(1.1)，就能把原本是浮點數的1.1轉換成整數形式的1。
    這邊我整理一下你們現在應該會常用的幾種。

    1. 數字

      2. 數字分為int(integer整數)、float(floating-point number浮點數)和complex(complex number複數)。這邊因為複數沒辦法對應相應的實數，所以先主要討論整數和浮點數的部分。
      我以10為例，整數形式int(10)為10，而浮點數形式(10)為10.0，接著以10.6為例，整數形式int(10.6)為10而浮點數float(10.6)為10.6。初學者容易犯的錯就是把兩個整數做運算後，得出的結果並非自己所想的。範例如下:

      程式碼
      print 3/6

      顯示為
      0

      一般而言雖然我們知道3/6等於0.5，但在這邊程式認知你的運算都是整數的類型，因此他把答案0.5取整數，就是程式所顯示的0了。解決的方法很簡單，只要把運算改成3.0/6或3/6.0或是3.0/6.0就能顯示0.5了。

    2. 數列與矩陣

      3. 可以處理數列的形式分為list和array這兩種，之前用np.arange叫出來的就是array的形式(雖然Type裡寫float64或int64，不過這之後再說)。這兩者都是儲存多個資料的方式，建立list的方法只要打上a=[1,2,3]或是a=range(10)就是一個list了!另一方面，array的話，幾乎用numpy建立出來的數列或矩陣都是這個形式的資料類型。

    3. 字串string


      4. 字串就是用來儲存文字資料的一種資料形式，使用方法只要在要儲存的文字兩側放上「'」即可，如「'文字'」

      範例: print 'hello world!'

  • 迴圈

    • 迴圈包含很多種，通常是用在需要重複做某一組指令，或是符合條件才執行指令時使用。而在python裡，迴圈只有開始的指令，沒有end之類的指令給他知道，迴圈要在哪裡跑。因此，他是用4個空白鍵(或是按一下Tab鍵)，來決定你迴圈裡的內容要執行哪幾行。
    這禮拜我只提for迴圈，所以我先打for迴圈就好。

    for迴圈是拿來執行固定次數的一個迴圈，執行的次數取決於你給的數列的長短。範例如下:

    程式碼
    a=[1,3,5,7,9]
    for i in a:
        print i
        print 'next'

    輸出為
    1
    next
    3
    next
    5
    next
    7
    next
    9
    next

  • 判斷條件(True or False)

    • 判斷條件是通過判斷一個或多個語句的結果為True或False，來決定執行的程式碼。判斷式中大概包含等於(==)、不等於(!=)、大於(>)、小於(<)和大於等於(>=)等種類。
    當符合判斷式條件的時候，Python會給出True的回應，反之則為False。大家可以用以下程式碼進行嘗試

    程式碼
    print 1==1
    print 1!=1

    輸出為
    True
    False

    另外如果有多個條件的話，則可以使用聯集(or)或是交集(and)的概念，範例如下

    print 1==1 or 1!=1

    結果為
    True

  • 條件語句(if)

    • 條件語句簡單講就假設一些條件，符合條件就執行code，不符合就不執行。設定的條件可以不只一個，以下做個範例

    程式碼
    a=[1,2,4,5,3,6]
    for i in a:
        if i<2:
            print str(i)+' matches condition1'
        elif i<4:
            print str(i)+' matches condition2'
        else:
            print str(i) +' other'

    輸出為
    1 matches condition1
    2 matches condition2
    4 other
    5 other
    3 matches condition2
    6 other

    這邊特別注意到其實在a數列中1是同時符合i<2以及i<4的條件，但是這邊程式只顯示符合i<2的結果。這是因為我們設定i<4這個條件是用elif，也就是else if，也就是不符合前面條件且符合elif設定的條件的時候就執行code，如果前面條件符合的話就不執行，有點類似差集的概念。而如果有要顯示「1 matches condition2」的話，就把elif i<4:改為if i<4:就行了。

  • 條件循環(while)

    • while語句使用上有點像條件語句結合for 迴圈。以下為範例

    程式碼
    i=0
    while i<3:
        print i
        i=i+1

    輸出為
    0
    1
    2

    這組code表示的是當條件i<3符合的時候，就執行print 和i=i+1，一直執行到條件不符合i<3的時候就跳出while之外。

  • 畫圖(matplotlib)

    • 這邊我主要介紹一些基本的畫圖法
    首先引入一個函式庫

    import matplotlib.pyplot as plt

    接著如果要畫一個2D圖的話，我們需要先建立兩個長度相同的數列，如

    x=[1,2,3,4]
    y=[4,5,2,3]

    這兩個數列代表著2D圖上的每一個x座標與y座標，跟excel畫圖有類似。畫圖方式如下

    plt.plot(x,y,'-o',label="note")
    plt.legend()

    呈現的結果如附圖


    第一條指令裡的第一和第二個元素代表x座標與y座標，第三個元素代表畫線的方式是用圓圈標示點並用線相連，第四個元素代表這條線的標籤(圖中右上角那個框框中的東西)。 而為了要在圖中建立右上角那個框框，我們需要使用第二條指令來建立 接著如果需要為座標軸標上單位的話，可以使用以下指令

  • List和Array的使用差異


    • 乘法對於list和array來說有不同的意義。對於list而言，乘法是對自己本身的數列重複幾次的指令，如[1,2,3]這個list去*2的話，就會形成[1,2,3,1,2,3]的list。另一方面，array所形成的數列的乘除法則比較相近於一般認知的乘除，也就是將數列出的每個元素都進行乘除法。以下為範例

    程式碼
    import numpy as np
    list=[1,2,3]
    array=np.arange(0.0,1.21,0.3)
    list*=3
    array*=3
    print 'list='+str(list)
    print 'array='+str(array)

    輸出為
    list=[1, 2, 3, 1, 2, 3, 1, 2, 3]
    array=[ 0. 0.9 1.8 2.7 3.6]

  • List使用上的問題


    • 這東西有點不太好說明。基本上算是程式本身設計上的一些缺陷，但python同時也有提供相應的解決方法，所以倒也不是什麼大問題。以下為範例

    程式碼
    a=[1,2,3]
    b=a
    b[1]=5
    print 'a='+str(a)+'\n'+'b='+str(b)+'\n'+'---'
    a=[1,2,3]
    c=a[:]
    c[1]=6
    print 'a='+str(a)+'\n'+'c='+str(c)

    輸出為
    a=[1, 5, 3]
    b=[1, 5, 3]
    ---
    a=[1, 2, 3]
    c=[1, 6, 3]

    如果再編寫程式碼的時候，直接將b=a的話，python會把b和a連結為相同的東西，因此在第四行，我將b的第1個元素更換為5的時候，a的第一個元素也會被跟著替換掉。
    因此，當你想要複製a的資料，但不希望在更動複製的資料時，變更a的資料時，可以使用像c=a[:]這樣的方法來複製。

  • 建立自訂方程式(def)


    • 自訂方程式其實算是滿好用的一個指令，如果在程式碼中有一很常用到的計算式的話，就能利用def讓整個程式碼看起來更簡潔。以下為範例


    import matplotlib.pyplot as plt
    import numpy as np
    def velocity(x1,x2):
        y=x1*x2
        return y
    t=np.arange(0.0,10.0,0.2)
    v=velocity(9.8,t)
    plt.plot(t,v,'-o')

    輸出的結果會是一張v=at的圖，這邊我稍微講解一下define的使用。方程式的話，一般較熟悉的莫過於f(x)=y，意思就是給定(input)一個值x，經過f()這個方程式轉換後，形成一個y的值。上面範例中v=velocity(9.8,t)的意思就是「將9.8和t分別代入velocity(x1,x2)中的x1和x2，接著經過方程式中的y=x1*x2計算後得出一個y值，再來就是把這個得出的y值回傳給v=velocity(9.8,t)這條方程式中的v」
